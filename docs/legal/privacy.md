# 隐私政策

最后更新: 2026-07-01

## 1. 数据收集

本 Bot **不主动收集**个人数据。在运行过程中可能临时处理以下信息：

### 自动处理的数据

| 数据类型 | 用途 | 存储方式 |
| --- | --- | --- |
| 用户 ID / 用户名 | 权限验证、配置名单匹配 | `config.yaml` (由服务器管理员配置) |
| 消息内容 | 批量清理（按条件过滤） | 不存储，仅处理中读取 |
| 频道 ID | 锁定计划、订阅通知 | `schedules.yaml` / `subscribe.yaml`（仅 ID，不含内容） |

### 日志

Bot 运行日志可能包含用户 ID 和指令调用记录，用于调试和审计。日志文件仅存储在运行机器本地，不会上传或分享给第三方。

## 2. 数据存储位置

所有数据存储在 Bot 运行的服务器本地文件中（`config.yaml`、`perm.yaml`、`schedules.yaml`、`subscribe.yaml`）。本 Bot 不使用任何外部数据库服务。

## 3. 数据共享

本 Bot **不会**将任何数据分享给第三方。

## 4. 数据删除

- 通过 `/perm rm` 可删除动态权限规则中的用户 ID
- 通过 `/lock unplan` 可删除计划锁定中的频道 ID
- 服务器管理员可从 `config.yaml` 中移除用户信息
- 如需完全清除日志，请删除 `logs/` 目录

## 5. 第三方服务

本 Bot 的[表情模块](/modules/emoji)会从配置的 `base_url` 拉取表情数据。请求不包含用户个人信息。

## 6. Cookie 与追踪

本 Bot 不使用 Cookie，不进行任何形式的用户追踪。
