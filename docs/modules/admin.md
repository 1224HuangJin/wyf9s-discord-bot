# 管理指令 (admin)

提供指令同步与模块热重载功能。

- **配置键**：无（始终加载）
- **源文件**：`cogs/admin.py`

## 指令

### `/sync` — 同步斜杠指令

向 Discord 同步斜杠指令列表（注册 / 更新命令树）。

| 项目 | 说明 |
| --- | --- |
| 权限 | Admin（仅 config admins） |
| 审计 | ✅ 记录 |

### `/reload` — 热重载模块

在不重启 bot 的情况下热更新单个 Cog 模块。

| 项目 | 说明 |
| --- | --- |
| 权限 | Admin（仅 config admins） |
| 参数 | `module`（可选，留空列出所有 Cog 及加载状态） |
| 冷却 | 15s / 用户 |

- 语音连接、限速状态、计划锁数据存储在 bot 实例上，重载不丢失
- 不带参数时列出 `cogs/` 下所有模块（绿圈 = 已加载，黑圈 = 未加载）
