# 管理指令 (admin)

提供指令同步与模块热重载功能。

- **配置键**：无（始终加载）
- **源文件**：`cogs/admin.py`

## 指令

### `/sync` — 同步斜杠指令

向 Discord 同步斜杠指令列表（注册 / 更新命令树）。

| 项目 | 说明 |
| --- | --- |
| 权限 | Admin（仅 config admins） |
| 审计 | ✅ 记录 |

### `/reload` — 热重载模块 / 配置

在不重启 bot 的情况下热更新模块或配置。

| 项目 | 说明 |
| --- | --- |
| 权限 | Admin（仅 config admins） |
| 参数 | `module`（可选，见下方行为） |
| 冷却 | 15s / 用户 |
| 审计 | ✅ 记录（`reload` / `reload-all` / `reload-config`） |

`module` 参数支持**动态自动补全**（从 `cogs/` 目录实时列出，结果缓存 1 秒），并有以下行为：

| 取值 | 行为 |
| --- | --- |
| 留空 | **重载全部**已加载模块 |
| `config` | **重载配置**：重新读取 `config.yaml` 并重新加载所有模块以生效 |
| `<模块名>`（如 `lock`） | 重载指定单个模块；若该模块当前未加载则尝试加载 |

- 语音连接、限速状态、计划锁数据、语言偏好等存储在 bot 实例上，重载不丢失。
- 重载 `perm` 模块（或全部 / 配置）会一并重新加载 `perm.yaml` 动态权限数据。
- `config` 重载仅对**已加载**的模块生效；若配置由「禁用」改为「启用」新模块，请用 `/reload <模块名>` 单独加载或重启。
